📂 SRM Multi-Threshold Sweep – Neuron 373 Intervention vs Baseline

This document provides a comprehensive technical breakdown of a multi-threshold SRM (Spotlight Resonance Mapping) experiment comparing clamped vs unclamped activation conditions for Neuron 373 in GPT-2 Small. All data were collected from Layer 11 MLP post-activations (hook_mlp_post) and analyzed using cosine similarity projections across a 2D spotlight plane defined by Neurons 373 and 2202.

🧪 Data Collection Pipeline

Prompts were drawn from two input sources: promptsv3.txt (general reasoning/language) and promptsv4.txt (targeted Neuron Viewer phrases)【34†v3】【35†v4】.

Baseline Capture

  • Script: capture_baseline_mlp_post.py【31†source】
  • Model: GPT-2 Small, no intervention
  • Each prompt generated 50 tokens (top-k sampling optional)
  • Captured at hook_mlp_post in Layer 11, dimension 3072
  • Mean-pooled vectors stored as results_v3/v4_no_intervention_mean_vectors_3072d.npz

Intervention Sweep

  • Neuron 373 was clamped from −20 to +20 using patched forward hooks
  • Otherwise identical pipeline to baseline
  • Vectors saved as results_v3/v4_len50_mean_vectors_3072d.npz

📐 SRM Analysis Method

The SRM sweep was implemented in run_srm_sweep_multi_threshold.py【33†source】, an enhanced version of a basic SRM analyzer (run_srm_sweep.py)【32†source】. This version supports sweeping cosine similarity thresholds at each angle.

Each vector was projected against spotlight directions defined by:

spotlight_vec = cos(θ) * e_373 + sin(θ) * e_2202

For each of 360 degrees:

  • Mean Cosine Similarity across all vectors (grey dashed line)
  • Threshold counts for similarities > 0.7, 0.5, 0.3, and 0.1

📊 Visual Results

Figure 2: Baseline (No Intervention)

🧠 Interpretation Summary

Direct Effects

  • Clamping Neuron 373 creates strong alignment with ±e_373 (seen in threshold spikes).
  • This confirms causal vector entrainment via intervention.

Second-Order Effects

  • Even without direct vector alignment to e_2202, the entire latent field rotates slightly toward −e_2202 (~270°).
  • Suggests Neuron 373 modulates not only local directionality, but also latent geometry of nearby activations.

Sparsity

  • Baseline space shows high sparsity — essentially no threshold crossings at even 0.1 similarity.
  • This underscores the effect of intervention in breaking alignment silence.
